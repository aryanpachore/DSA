## 🛎️ How to Use

1. **Clone the Repository**
    ```bash
    git clone git@github.com:aryanpachore/DSA.git
    ```

2. **Navigate to a Topic**
    ```bash
    cd DSA/Arrays
    ```

3. **Run the Code (C++)**
    ```bash
    g++ filename.cpp -o output
    ./output
    ```
